# Calculadora
Proyecto personal , Esta hecho con HTML5 , CSS3 ,Sass y Javascritp (EM6).

## Descripción 

hace operaciones basicas:
- Sumar
- Restar
- Multiplicar
- Dividir
- Borrar
- C
- CE

## Uso
clickea en los botones y has las operaciones que necesites.
